# food-recipe-app

-- In order to run the app, first fork the project then clone it using cmd. 

-- Open the folder in the vscode, then you open terminal, then type" npm install" , 
   that will install all the node modules. 

-- After installation is done, type "expo start". 

-- The metro bundler will be open in your default web browser, then you click on either 
   run on iOS simulator, android simulator or web. 

-- you can also scan the QR code with your iOS camera to run the app on your phone.
